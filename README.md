# MLDS_hw2
## Hw2-1
### Prerequisites
    $ pip3 install tensorflow
    $ pip3 install numpy
    $ pip3 install tqdm
    $ pip3 install json

### Installing
    $ cd hw2-1/
    $ ./run.sh

### Format
    > hw2-1
        > MLDS_hw2_1_data/
        > model/
        > run.sh
        > seq2seq.py
        > data_processor.py
        > train.py
        > infer.py

### Prerequisites
